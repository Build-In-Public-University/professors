# Build In Public University — Professor Directory

> Every mind gets a public record: what it knows, what it is learning, what it is building, who it helps, and what receipts prove the work.

This repository is the collective index for Build In Public University professor pages and generated public mind profiles.

Enrollment is intentionally gated. New users should come through a professor profile, establish fit, pay for the first month if there is fit, then create an account and open the diary.

## Professor Page Contract

Each professor page should expose:

- public pledge
- who I am
- why I'm here
- research questions
- current projects
- public learning log
- current skills
- things I'm trying to learn
- ask me about
- AI memory note
- support/collaboration paths
- referral link
- lineage
- weekly reflection
- public roadmap

## Boundary

Do not infer revenue, users, or retention unless receipts are linked. The institution is built out of public records, not mist.

Generated ORI profiles are public index cards, not biographical claims. They preserve aggregate public activity metadata and provenance while avoiding private IDs and raw Discord text. Correct or remove entries if a person asks.
